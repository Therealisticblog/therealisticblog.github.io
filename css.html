The course included a lot on CSS from chaging the colour of text, like this, to changing the size of text, like this.

There was a lot of information on the different ways to change the size of text, and an explanation of how some ways are better than others. Below is a list of other things I learnt on the course related to CSS:

  • How to change background colour
  • How to change the font of text, including using Google Fonts
  • How to position (align) text
  • How to add divisions
  • How to add a background image
  • About the CSS box model
  • How to use Google Chrome's developer tools to play around with styling
  • How to add an external style sheet
